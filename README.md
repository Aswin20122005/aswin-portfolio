# Aswin's Portfolio

A premium, dark-themed developer portfolio with glassmorphism, particle effects, animated counters, project filtering, and more.

## Quick Deploy to Vercel (Free)

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [Git](https://git-scm.com/)
- A [GitHub](https://github.com) account
- A [Vercel](https://vercel.com) account (sign up free with GitHub)

### Step 1: Install dependencies
```bash
cd aswin-portfolio
npm install
```

### Step 2: Test locally
```bash
npm start
```
Opens at http://localhost:3000

### Step 3: Push to GitHub
```bash
git init
git add .
git commit -m "My portfolio"
git branch -M main
git remote add origin https://github.com/Aswin20122005/aswin-portfolio.git
git push -u origin main
```

### Step 4: Deploy on Vercel
1. Go to https://vercel.com and sign in with GitHub
2. Click "Add New" → "Project"
3. Import your "aswin-portfolio" repo
4. Click "Deploy"
5. Done! Your site is live.

## Tech Stack
- React 18
- Lucide React (icons)
- CSS animations & IntersectionObserver
- Canvas particle system
